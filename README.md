Vehicle Service Management System

Overview
This project is a web-based Vehicle Service Management System developed using HTML, CSS, JavaScript, PHP, and MySQL.

The system uses a relational database with the following tables:
- Users
- Vehicles
- Services
- Mechanics
- Service Requests
- Payments

SQL Features Used
- JOIN queries
- GROUP BY and aggregate functions
- Foreign key constraints
- Relational schema design

Sample Queries
- Fetch service requests with user and vehicle details
- Calculate total revenue
- Track service status

Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
